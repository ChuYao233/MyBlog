# 友链添加说明

要添加友链，只需在此目录下创建一个新的 YAML 文件即可。

## YAML 文件格式

文件名可以是任意名称（建议使用友链名称的拼音或英文），例如：`example.yaml`

文件内容格式：

```yaml
name: 友链名称
url: https://example.com
description: 友链描述（可选）
avatar: https://example.com/avatar.jpg（可选）
```

## 必需字段

- `name`: 友链名称
- `url`: 友链网址

## 可选字段

- `description`: 友链描述
- `avatar`: 头像图片 URL
- `email`: 联系邮箱
